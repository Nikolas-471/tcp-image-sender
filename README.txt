Как использовать?
1. Запустите файлы из папки release, client.exe и server.exe на машине клиента и сервера соотвественно.
2. Введите IP сервера в приложении клиента в строку Server name
3. Введите порт сервера в приложении клиента в строку Server port
4. Нажмите кнопку Get Server, убедитесь что соединение произошло и в строке-информаторе клиента отобразилась информация об этом. 
   В ином случае проверьте правильность IP и порта исходящего компьютера, а так же разрешения программы для работе в сети.
5. Нажмите кнопку Select image и выберите изображение для отправки на сервер, убедитесь что изображение отобразилось на стороне клиента.
6. Нажмите кнопку Send image и убедитесь что изображение отобразилось на стороне сервера.
